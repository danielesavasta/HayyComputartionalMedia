# Media Anatomy

Every digital media object (image, sound, video, interactive system) shares the same underlying structure. Understanding that structure is the first step toward working with it intentionally.

:::

::: swiper-slide

##### Foundation Principles

# Numerical Representation

Every media object consists of digital code: mathematically definable, open to algorithmic manipulation. Media becomes programmable.

A photograph is a grid of numbers. A sound is a sequence of values. This numerical nature means any media object can be transformed, generated, or combined algorithmically.

:::

::: swiper-slide

##### Foundation Principles

# Procedurality

The procedure (the algorithm, the instruction set) is often the artwork itself, not just the means of making it.

You write the rules; the rules write the world. The artist is the author of conditions, not of outcomes.

:::

::: swiper-slide

##### Foundation Principles

# Crossing Senses / Intermediality

The same data moves between mediums: sound becoming light, movement becoming image, text becoming space.

Media is not locked to one sense or form. Intermediality is letting one data source speak in multiple languages at once.

:::

::: swiper-slide

##### Foundation Principles

# Co-existence

The simultaneous presence of analog and digital, physical and virtual, human and nonhuman within the same system or space.

Co-existence is about designing the relationship between these registers: how they touch, depend on, and transform each other.

:::

::: swiper-slide

##### Properties

# Generativity / Variation

How changing one variable produces an entire family of related but distinct forms, sounds, or behaviors.

The artist designs the conditions; the system generates the instances. Variation is controlled difference within a defined space of possibility.

:::

::: swiper-slide

##### Properties

# Modularity / Repetition

A small repeating element, when scaled and layered, becomes something structurally new.

The module retains its identity while contributing to a larger whole. At scale, repetition produces emergence. What was one thing becomes a field, a texture, an environment.

:::

::: swiper-slide

##### Properties

# Connectivity / Network

How devices, sensors, spaces, and people share information and collectively respond to the world.

A connected system is not the sum of its parts. It is the sum of its relationships. New behaviors emerge that no single component could produce alone.

:::

::: swiper-slide

##### Properties

# Limits

Origins. Constraints. Dataset. Pool. Source.

Every system is defined as much by what it cannot do as by what it can. Limits are not obstacles. They are the edges that give the work its character.

:::

::: swiper-slide

##### Process

# Profanation / Appropriation

Taking something from one context and returning it to common use in a different one. Not destruction. Displacement.

The same material, made to mean something else.

:::

::: swiper-slide

##### Process

# Edit / Abandon

The generative process produces more than you intend. At some point: intervene and edit, or abandon and let the system run again.

This is not failure. It is the practice. The decision defines the relationship between the artist and the system.

:::

::: swiper-slide

##### Process

# Randomness / Chaos / Accident / Serendipity / Letting Go / Entropy

The system produces things you did not plan. Some are more interesting than anything you could have planned.

Accident is not the enemy of intention. Letting go is the practice of remaining open to what the system discovers on its own.

:::

::: swiper-slide

##### Meaning Making

## Text as Meaning / Poetic Computation

Creative coding. Vibe-coding. LLMs.

Language can be the material the work is made of, not just a description of it. Code is writing. Prompts are composition. Computation becomes a poetic act.

:::

::: swiper-slide

##### Meaning Making

## Interaction Ecology

How human-nonhuman-machines interact within a frame or space, and how their roles alter in relation to each other.

Not a single user pressing a button. A field of bodies, sensors, outputs, and histories in simultaneous relation.

:::

::: swiper-slide

##### Meaning Making

## Lore

Context. Artistic Intention.

What separates "a grid of blinking LEDs" from "a field of breathing organisms." The references a piece carries, the world it implies, the intention that makes formal choices feel deliberate rather than arbitrary.

:::

::: swiper-slide

##### Tools

## p5.js / Processing

Code-based environments for visual, sonic, and interactive work. Both treat the screen as a sketchbook for rapid iteration.

You write a rule, you see what it makes. The most direct path from an algorithmic idea to a visible result.

:::

::: swiper-slide

##### Tools

## Sensors / Hardware / Software / Network

Microcontrollers, sensors, actuators, and the protocols that connect them: OSC, MQTT, WebSocket.

This is where the system touches the world. A sensor reads the environment; software decides; an actuator acts.

:::
